Input Mask

A plugin to make masks on form fields.

Input Masks documentation

How to use

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="@@autopath/node_modules/jquery-mask-plugin/dist/jquery.mask.min.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

              
                <script src="@@autopath/assets/js/hs-mask.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // INITIALIZATION OF INPUT MASKS
                    // =======================================================
                    $('.js-masked-input').each(function () {
                      $.HSCore.components.HSMask.init($(this));
                    });
                  });
                </script>

Selected On Focus

Use "selectOnFocus": true

  • Preview
  • HTML 
                    
                      <input type="text" class="js-masked-input form-control" placeholder="xx/xx/xxxx"
                            data-hs-mask-options='{
                               "template": "00/00/0000",
                               "selectOnFocus": true
                            }'>

Autoclear

Use "clearIfNotMatch": true

  • Preview
  • HTML 
                    
                      <input type="text" class="js-masked-input form-control" placeholder="+x(xxx)xxx-xx-xx"
                            data-hs-mask-options='{
                              "template": "+0(000)000-00-00",
                              "clearIfNotMatch": true
                            }'>

Custom Pattern

  • Preview
  • HTML 
                    
                      <input type="text" class="js-masked-input form-control" placeholder="A-**-000-a000"
                            data-hs-mask-options='{
                              "template": "A-**-000-a000",
                              "translation": {
                                "*": {"pattern": "[A-Za-z0-9]"}
                              }
                            }'>

                          });
                        });
                      </script>
