jVector Map

Many maps of the world, world regions, countries and cities.

jVector Map documentation

How to use

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="@@autopath/assets/vendor/jvectormap/jquery-jvectormap-2.0.3.min.js"></script>
                <script src="@@autopath/assets/vendor/jvectormap/tests/assets/jquery-jvectormap-world-mill-en.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

              
                <script src="@@autopath/assets/js/hs.jvector-map.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // initialization of jvectormap
                    $('.js-jvectormap').each(function () {
                      var jVectorMap = $.HSCore.components.HSJVectorMap.init($(this));
                    });
                  });
                </script>
