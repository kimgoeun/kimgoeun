Password Strength

jQuery Password Strength Meter.

Password Strength documentation

How to use

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="@@autopath/node_modules/pwstrength-bootstrap/dist/pwstrength-bootstrap.min.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

              
                <script src="@@autopath/assets/js/hs-pwstrength.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // INITIALIZATION OF PASSWORD STRENGTH MODULE
                    // =======================================================
                    $('.js-pwstrength').each(function () {
                      var pwstrength = $.HSCore.components.HSPWStrength.init($(this));
                    });
                  });
                </script>

Password requirements:

Ensure that these requirements are met:

  • Minimum 8 characters long - the more, the better
  • At least one lowercase character
  • At least one uppercase character
  • At least one number, symbol, or whitespace character
