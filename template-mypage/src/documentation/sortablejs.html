SortableJS

SortableJS JavaScript library for reorderable drag-and-drop lists.

SortableJS documentation

How to use

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="@@autopath/node_modules/sortablejs/dist/sortable.umd.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

              
                <script src="@@autopath/assets/js/hs.sortable.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // INITIALIZATION OF SORTABLE
                    // =======================================================
                    $('.js-sortable').each(function () {
                      var sortable = $.HSCore.components.HSSortable.init($(this));
                    });
                  });
                </script>

                      <script>
                        $(document).on('ready', function () {
                          // INITIALIZATION OF SORTABLE LINK
                          // =======================================================
                          $('.js-sortable-link').click(function (e) {
                            if ($(e.target).hasClass('hs-unfold') || $(e.target).parents('.hs-unfold').length) {
                              return false;
                            }

                            window.location.replace($(this).attr('data-href'))
                          });
                        });
                      </script>
