Count Characters

Show count characters in the field.

How to use

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

              
                <script src="@@autopath/assets/vendor/hs-count-characters/dist/js/hs-count-characters.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

              
                <script>
                  $(document).on('ready', function () {
                    // INITIALIZATION OF COUNT CHARACTERS
                    // =======================================================
                    $('.js-count-characters').each(function () {
                      new HSCountCharacters($(this)).init()
                    });
                  });
                </script>
